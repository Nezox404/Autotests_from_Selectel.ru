## Тест-кейсы

### Главная страница
- Проверка заголовка главной страницы

### Документация
- Переход по ссылке "Документация"
- Проверка URL открывшейся страницы
- Проверка заголовка страницы документации

### Поиск продуктов
- Поиск продукта "Кластеры Kubernetes"
- Проверка наличия результатов поиска
- Переход на страницу продукта
- Проверка хлебных крошек
- Проверка заголовка страницы продукта

### Негативные сценарии
- Поиск по несуществующему запросу
- Проверка отсутствия результатов
- Проверка сообщения "Мы ничего не нашли по вашему запросу"

## Установка и запуск

### Установка зависимостей
```
pip install -r requirements.txt
```

### Запуск всех тестов
```
pytest
```

### Запуск smoke тестов
```
pytest -m smoke
```

### Запуск с формированием Allure отчета
```
pytest --alluredir=allure-results
allure serve allure-results
```
